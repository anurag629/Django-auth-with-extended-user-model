# Django-auth-with-extended-user-model 

## 1. Create project and app :
* create project:

        django-admin startproject core
* create app inside the project main directory:

        python manage.py startapp accounts

## 2. Add app to settings.py file :
    INSTALLED_APPS = [
        ....,
        'accounts',
        ....,
    ]
